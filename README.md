# serieztv-api
API for serieztv mobile app

## Structure

- src
    - models
    - views
    - controllers
    - `app.js`
 
## Requirements

- node.js (`^6.9.1` preferred)
- ES6 for development environment 

## Installation

install all dependencies from package.json

    npm install
    
## Run

For development

    npm run dev
    
For production

    npm start
